## LeSingîla - Site Web d’un Restaurant Fictif

# Description

LeSingîla est un site web fictif conçu pour un restaurant.
Ce projet a pour objectif de créer une interface élégante et fonctionnelle en utilisant uniquement HTML et CSS.

# Fonctionnalités

- Page d’accueil : Présentation du restaurant avec une brève description et des images attractives.
- Menu : Liste des plats proposés avec descriptions et prix.
- Galerie : Collection d’images mettant en avant l’ambiance et les plats du restaurant.
- Contact : Informations de contact et formulaire de prise de contact.
- Réservation : Formulaire permettant aux clients de réserver une table.

# Technologies utilisées

- HTML5 : Structure du site.
- CSS3 : Mise en page et design.
